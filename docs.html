Requirements

At the top of every document, you must include <!DOCTYPE html> to make sure you get all modern futures. The end product will look something like this:

	
	<!DOCTYPE html>

	<html>

	<head>

	    <meta charset="utf-8">

	    <title>Framework Template</title>


	    <link rel="stylesheet" href="css/framework.css">

	</head>

	<body>

	</body>

	</html>
